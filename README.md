# blogmate


## Windows
```
git clone https://github.com/taeho0888/blogmate.git .
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## MacOS
```
git clone https://github.com/taeho0888/blogmate.git .
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
uvicorn main:app --reload
```
